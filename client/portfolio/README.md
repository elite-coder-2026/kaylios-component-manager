# Portfolio (static page)

Open `client/portfolio/index.html` in a browser.

Edit content in `client/portfolio/index.html` inside the `<script id="portfolioData" type="application/json">` block:

- `name`, `role`, `bio`, `location`, `email`
- `links.github`, `links.linkedin`, `links.resume`
- `projects[]` and `skills[]`
- Optional project previews: set `projects[].image` to a relative path (ex: `./img/my-project.png`) and `projects[].imageAlt`.
